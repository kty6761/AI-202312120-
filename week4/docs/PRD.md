# PRD — Week 4 Physics Neural Networks GUI

**버전:** 1.0
**날짜:** 2026-04-06
**대상:** AI/ML 과목 수강생

---

## 1. 개요

물리 데이터를 학습하는 Neural Network 4개 실습을 단일 GUI 앱에서 실행·시각화할 수 있는 데스크탑 애플리케이션.

---

## 2. 사용자 목표

- 코드를 수정하지 않고 파라미터를 바꾸며 신경망 학습 결과를 즉시 확인한다.
- 학습 결과를 그래프로 보고 PNG 파일로 저장한다.
- 4개 실습(1D 근사, 포사체, 오버피팅, 진자)을 하나의 창에서 전환하며 비교한다.

---

## 3. 기능 요구사항

### 3.1 공통
| ID | 요구사항 |
|----|---------|
| F-01 | 단일 창, 탭으로 4개 Lab 전환 |
| F-02 | 각 탭: 파라미터 입력 → 학습 시작 버튼 → 그래프 표시 → PNG 저장 |
| F-03 | 학습 완료 후 `outputs/` 폴더에 PNG 자동 저장 |
| F-04 | 하단 로그 영역에 학습 진행 상태 출력 |

### 3.2 Lab 1 — 1D 함수 근사
| ID | 요구사항 |
|----|---------|
| F-11 | 함수 선택: sin(x), cos(x)+0.5sin(2x), x·sin(x) |
| F-12 | Epochs, Hidden Layers, Learning Rate 입력 가능 |
| F-13 | 실제값 vs 예측값 곡선 비교 그래프 출력 |

### 3.3 Lab 2 — 포사체 운동
| ID | 요구사항 |
|----|---------|
| F-21 | 초기속력(v₀), 발사각(θ) 입력 가능 |
| F-22 | 실제 궤적 vs NN 예측 궤적 비교 그래프 출력 |

### 3.4 Lab 3 — 오버피팅 데모
| ID | 요구사항 |
|----|---------|
| F-31 | Epochs, 노이즈 크기 입력 가능 |
| F-32 | Underfit / Good Fit / Overfit 3개 모델을 동일 그래프에 비교 출력 |

### 3.5 Lab 4 — 진자 주기 예측
| ID | 요구사항 |
|----|---------|
| F-41 | 진자 길이(L), 초기 각도(θ₀) 입력 가능 |
| F-42 | 주기 예측 그래프(RK4 실제 vs NN 예측) + RK4 시뮬레이션 그래프 출력 |

---

## 4. 비기능 요구사항

| ID | 요구사항 |
|----|---------|
| NF-01 | 창 최소 크기 1000×700 px |
| NF-02 | 그래프 zoom/pan 인터랙션 지원 |

## 5. 범위 외 (Out of Scope)

- 학습 중 실시간 loss 업데이트 (학습 완료 후 결과만 표시)

- 다중 실험 결과 비교
